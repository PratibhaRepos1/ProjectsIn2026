# AI Projects

Overview
- Purpose: A workspace for small AI-related experiments, exercises, and learning notes.
- Location: This folder contains Python samples, notes, and subfolders for experiments.

Contents
- `Python Code/` — runnable example scripts (e.g., `first.py`).
- `notes.md` — Day-by-day setup and learning notes (Day 1 guide included).

Quick Start (Windows)
1. Open this folder in VS Code: `code .`
2. Check Python: `python --version`
3. Create virtual env: `python -m venv venv`
4. Activate env: `venv\\Scripts\\activate`
5. Run example: `python "Python Code\\first.py"`

Developer tips
- Use the `notes.md` for step-by-step setup and next-day plans.
- Add dependencies to `requirements.txt` and install via `pip install -r requirements.txt`.
- Enable VS Code extensions: Python, Pylance.
